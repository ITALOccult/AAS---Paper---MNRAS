# figures/README.md
# Questa directory contiene le figure del paper in formato PDF.
#
# Le figure vengono generate dagli script in scripts/:
#
#   cd scripts/
#   pip install -r requirements.txt
#   python plot_all.py
#
# I file PDF sono inclusi nel repo (.gitignore ha l'eccezione
# per figures/*.pdf) in modo che Overleaf possa compilare
# il paper senza dover eseguire gli script.
#
# File attesi:
#   fig_energy_vs_precision.pdf    → Figura 1
#   fig_divergence_time.pdf        → Figura 2
#   fig_step_distribution.pdf      → Figura 3
#   fig_shadow_hamiltonian.pdf     → Figura 4
#   fig_stm_accuracy.pdf           → Figura 5
